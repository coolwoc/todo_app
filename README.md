# todo_app
GULP, SASS and Angular 1.3

	The user can register and login.
	The user can add tasks.
	They can edit and remove tasks.
	The app filters by :

		[] Task done.
		[] Task archived.
		[] Next 7 days tasks.
		[] Daily tasks.

	** Important ** 
	I have been using json-server. 
	https://github.com/typicode/json-server.
