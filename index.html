TO-DO app

This app has a custom adaptive layout and demonstrates my learning curve using some of the last technologies on the front-end side. Gulp ( a task automata for the front-end app ), SASS ( using modular architecture ) and ANGULAR ( with the primary use $resource and ui-router).

GULP

It is a build system to automate tasks. The file is in progress and DEPLOYMENT task has not been develop yet as I prioritise PRODUCTION tasks at the minute.

SASS

Applied SMACS, a modular architecture which allows to reuse CSS across the project.

ANGULAR

Followed John Papa style-guides to be able to write a more consistent and reusable abstract/OOP code.

GIT

All the stable code will be in MASTER. I will branch the next features of the app and merge them when they are stable enough.

THIRD PARTY

  • Font Squirell - Allows to embed fonts in css file with a good render without the need of using JS.
  • Iconic-font - Open source icons.
  • Json-server - Used to handle locally $resource request from db.json file.
