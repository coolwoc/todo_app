Hi folks!

This is a small application which you can achieve the following:

  • The user can add tasks
  • They can edit and removed tasks
  • The application filters by:
    • [] Task done.
    • [] Task archived.
    • [] Next 7 days tasks.
    • [] Daily tasks.

Some technical info:

I had worked on the layout and design of the app. However, img assets like logo have been grabbed from internet. The purpose of this application is not commercial. This is a pet-project.

I use GULP to automate the front-end tasks.

The architecture of the project is based on SMACS - a more scalabel and modular architecture for the SASS/CSS - and John Papa style guides for a more abstract/modular ANGULAR
