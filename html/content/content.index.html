Hi there!

This is a small todo-app I have develop in the last days. I will like to demostrate my knoledge of Angular using primary $resource and ui-router. Two of the most poweful features of the framework on my understanding. All the features of the app are custom or using the power of Angular as a framework.

I had worked on the layout and design of the app. However, img assets like logo have been grabbed from internet. The purpose of this application is not commercial is just merely educative. Therefore, I cannot see any problem doing that.

I use GULP to automate the front-end task, so I could experiment and have a more inside of it as I was more familiar to GRUNT. I have to highlight that the use of jshint has helped me lots to debbug the terrible way sometimes Angular handle their exceptions. There is not deployment task yet.

The architecture of the project is based on SMACS - a more scalabel and modular architecture for the SASS/CSS - and John Papa style guides for a more abstract/modular architecture for ANGULAR

Task in GULP still in development as there is a few improvements left and the app has a long list of functionality and modules that are not in place yet. I would provied the git-master, with the most stable code and will branch in order to carry with the app.

If you want to give feedback or have any questions please don't hesitate to contact by email.
